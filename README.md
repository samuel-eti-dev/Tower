# Tower
Tower est un jeu de stratégie au tour par tour développé en C. Deux joueurs s'affrontent sur une grille 6x6. Chacun construit des tours sur les cases du plateau. Une tour peut atteindre une hauteur maximale de 3 et exerce une domination sur les cases adjacentes. Le joueur qui contrôle le plus de cases à la fin de la partie remporte la victoire.
